
| Ableitung                              | Bedeutung           |
| -------------------------------------- | ------------------- |
| $f'(x)$                                | Steigung            |
| $f'(x)=0$                              | Extremalstelle      |
| $f''(x)$                               | Form der Parabel    |
| $f''(x)>0$                             | Nach oben geöffnet  |
| $f''(x)<0$                             | Nach unten geöffnet |
| $f'(x)=0 \text{ and } f''(x)>0$        | Lokales Minimum     |
| $f'(x)=0 \text{ and } f''(x)<0$        | Lokales Maximum     |
| $f''(x)=0 \text { and } f'''(x) \ne 0$ | Wendepunkt          |
