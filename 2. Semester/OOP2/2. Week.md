### Wildcards
`<? extends Animal>` = alle methoden die animals extenden
```Java
public static void printAnimals(List<? extends Animal> animals) {}
```

|               | Typ              | Kompatible Typ-Argumente | Lesen | Schreiben |
| ------------- | ---------------- | ------------------------ | ----- | --------- |
| Invarianz     | `C<T>`           | `T`                      | yes   | yes       |
| Kovarianz     | `C<? extends T>` | `T und Subtypen`         | yes   | no        |
| kontravarianz | `C<? super T>`   | `T und Basistypen`       | no    | yes       |
| bivarianz     | `C<?>`           | Alle                     | no    | no        |
